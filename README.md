# Verifica News

Este é um projeto de extensão de navegador para verificar a veracidade de notícias, usando a Google Fact Check API e a NewsData API.

## Como usar

1. Clone este repositório.
2. Abra o arquivo HTML em um navegador.
3. Insira uma palavra ou frase para buscar notícias e verificações de fatos.

## Tecnologias

- HTML
- CSS
- JavaScript
- Google Fact Check API
- NewsData API
